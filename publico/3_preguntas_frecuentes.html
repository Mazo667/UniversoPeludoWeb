PREGUNTAS FRECUENTES

¿Cual es el horario de apertura y cierre del Refugio?

  • El refugio abre sus puertas de lunes a viernes de 9:00 a 18:00 hs y los sábados de 9:00 a 13:00 hs.

    • ¿Cual es el proceso de adopcion de una mascota?

  • Debes registrarte en el sitio web para poder obtener la informacion detallada para la adopcion de una mascota.

    • ¿Entregan a los animales ya vacunados y/o castrados?

  • Si son animales jovenes o adultos, se entregan castrados y vacunados. En caso contrario, si son cachorritos, se programa una cita con el veterinario para cuando cumpla la edad necesaria para ser esterilizado.

    • ¿Puedo visitar el lugar antes de adoptar?

  • Si, puedes visitar el refugio en el horario de apertura y cierre.

    • ¿Que puedo hacer para ayudar pero sin adoptar ni ser voluntario?

  • Podes realizar donaciones de manera presencial. Las donaciones pueden ser camas, ropa, comida, entre otros, o incluso enviar un aporte economico a nuestro cbu o alias.

    • ¿Como hago para entregar un animal al Refugio?

  • Puede acercarse directamente al Refugio o tambien en caso de no tener movilidad, puede contactarse con nosotros para mandar a un personal a buscar al animal en cuestion.

    • ¿Como puedo obtener informacion detallada sobre un animal que me interesa?

  • Debe registrarse en el sitio web para ver la informacion detallada de cualquier animal.

    • ¿Que pasa si la adopcion no funciona y debo devolver el animal al refugio?

  • Cada vez que se realiza la adopcion, se lleva al cabo un seguimiento y en caso de no poder cuidar correctamente del animal adoptado, se programa una cita con el rescatista y se devuelve.
